## 📌 About the Project

This project focuses on predicting medical insurance costs based on user information such as age, BMI, smoking habits, and region.

The goal was to build a simple yet effective machine learning model and deploy it as an interactive web application so that anyone can use it in real time.

---

## 🧠 How it Works

* The dataset is first cleaned and explored to understand patterns
* Categorical features are converted into numerical form
* Multiple models are trained and compared
* Random Forest is selected as the final model
* The model is deployed using Streamlit for real-time predictions

---

## 📊 Features Used

* Age
* Gender
* BMI (Body Mass Index)
* Number of Children
* Smoking Status
* Region

---

## 🤖 Model Details

The final model used is **Random Forest Regressor**, as it performed better compared to other models in terms of accuracy and error metrics.

---

## 📈 Evaluation Metrics

* R² Score → Measures how well the model fits the data
* MAE (Mean Absolute Error) → Average prediction error
* RMSE → Penalizes larger errors
